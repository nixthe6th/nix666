## 1. New AI Models & APIs

### IQuest Coder V1 (China) — GAME CHANGER
- **Released:** January 1, 2026 by IQuest Lab (Ubiquant hedge fund AI research arm)
- **Size:** 40B parameters (vs 400-800B+ for Claude/GPT)
- **Performance:** BEATS Claude Sonnet 4.5 and GPT 5.1 Mini on coding benchmarks
  - SWE-Bench Verified: 81.4% (vs Claude's 81.3%)
  - LiveCodeBench v6: 81.1% (vs Claude's 80.4%)
  - BigCodeBench: 49.9% (vs Claude's 47.8%)
- **Variants:**
  - Instruct: Daily coding companion
  - Thinking: Complex reasoning/algorithmic (competitive programming)
  - Loop: Recurrent transformer, efficient deployment
- **Secret sauce:** "Code-Flow Training" — trained on commit histories, repo evolution, code transformations rather than static snapshots
- **API:** IQuest Lab working on hosted API access (no pricing yet)
- **Open source:** 7B, 14B, 40B sizes available, quantized versions for consumer GPUs

### Model Context Protocol (MCP) — The "USB-C for AI"
- **What:** Open standard by Anthropic (Nov 2024) now being adopted by Google & OpenAI
- **Purpose:** Standardized way to connect AI assistants to data systems, APIs, databases
- **Why it matters:** Eliminates custom integrations — any MCP-compatible agent can use any MCP-compatible tool
- **Adoption:** Thousands of MCP servers already demonstrating rapid ecosystem growth
- **Actionable:** OpenAI Agents SDK now supports MCP with approval workflows
